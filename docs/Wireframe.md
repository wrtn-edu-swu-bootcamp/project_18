## 6. 상호작용 플로우

### 6.1 재고 요청 플로우
```
[재고 요청 페이지] → [매장/제품/수량 입력] → [이메일 발송]
        ↓
[상대 매장 출고 대기] → [승인] → [배송] → [요청 매장 입고 대기] → [완료]
```

### 6.2 수선 관리 플로우
```
[수선 등록] → [수선 전 탭] → [수선 보내기 (시간 설정)]
        ↓
[수선 중 탭] → (타이머 자동 완료)
        ↓
[수선 완료 탭] → [결제 확인] → [전달 완료 체크] or [알림톡 발송]
```
